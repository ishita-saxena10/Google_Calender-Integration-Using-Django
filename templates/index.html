Convin Backend Assignment

Google Calendar integration using django rest api.

Know More

This Assignment is implemented in Django and uses Google Calender API.

API endpoints:

    Please visit the following endpoints to see the proper implementation of the project.
  • /rest/v1/calendar/init/ : This will start the first step of OAuth2 mechanism which includes authorization from the user using their google account.
  • /rest/v1/calendar/redirect/ : This saves token in a file ("token.json" in project) which is automatically created once the flow completets and then returns all the events in the users calender.
